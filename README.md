##      **Quick Notes for Users**   

Although most of the template can be used intuitively, there are some notes from the creator to keep in mind to fully enjoy the experience for a prolonged period of time. 

Instructions on how to use the specific aspects of the template:

* Courses  
  * For first-time use:  
    * change the course names to your semester's courses  
    * change the course names in key databases: Database with Courses, Database (with time), HW and Notes.  
  * For future courses:  
    * make a new course and set the status to Current, and set the old courses' status as Archive  
    * For new courses, select one of three templates: Course TEMPLATE (with labs), Course TEMPLATE (English/Literature), Course TEMPLATE (Regular)  
    * Update courses names in key databases: Database with Courses, Database (with time), HW and Notes.  
    * Update grades outlook Views and Subject Filters  
* Journal  
  * New pages should be created every day. Click on No dates at the top of the database to open today's journal template  
* Aspirations  
  * When updating progress for the habit tracker, enter the number of times the activity was done, a "/" to indicate division, and the total number of times the activity was planned to do in the given period of time  
  * Update Progress as you go  
* Reading List  
  * when updating the number of pages read, enter the number of pages read, a "/" to indicate division, and the total number of pages in the book
